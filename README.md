# Maven-Market-Power-BI-dashboard
**Sales** **|** **Profit** **|** **Returns** **|** **Regional Insights**

**Overview**
This project delivers a complete Power BI dashboard analyzing Maven Market’s monthly performance across sales, profit, and product returns. It’s designed to give leadership a clear, interactive view of how the business is performing across brands and regions.
The dashboard includes KPIs, brand performance metrics, regional revenue distribution, and weekly revenue trends, all built using a clean data model and DAX measures.

**Objectives**
- Track current month performance against goals
- Identify top performing brands and high return categories
- Compare regional revenue across the US, Mexico, and Canada
- Visualize weekly revenue trends
- Provide a single, executive‑ready view of business health

**Tools & Techniques**
- Power BI Desktop
- Power Query for cleaning and shaping data
- DAX for KPIs and calculated measures
- Star schema data modeling
- Interactive visuals including KPIs, matrix tables, maps, and bar charts

**Data Model**
The dashboard is built on a star schema with:
- Fact tables: Sales, Returns
- Dimension tables: Products, Stores, Regions, Calendar
This structure keeps calculations efficient and makes the model easy to scale.

**Key Insights**
- Monthly KPIs show strong performance across transactions and profit
- Mexico contributed a significant share of revenue this period
- Certain brands show higher return rates, signaling potential quality or logistics issues
- Weekly revenue spikes align with promotional periods

**Business Impact**
This dashboard helps stakeholders:
- Monitor performance in real time
- Identify underperforming brands or regions
- Improve product quality and reduce returns
- Make data‑driven decisions about promotions and inventory
